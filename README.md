# Bien
This is one of the projects from 8-week online Rails course by [Superhi](https://www.superhi.com/courses/ruby-on-rails)

How to run
----

1. Clone this repo
2. Run `bundle install`
3. Run `db:create`
4. Run `db:migrate`
5. Run `rails server`
6. Visit `http://localhost:3000/`
